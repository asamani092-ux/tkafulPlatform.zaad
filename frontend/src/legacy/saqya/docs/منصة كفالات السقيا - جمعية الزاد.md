# منصة كفالات السقيا - جمعية الزاد
## ملخص المشروع الشامل

### نظرة عامة
تم تطوير منصة إلكترونية متكاملة لإدارة كفالات السقيا في جمعية الزاد، تهدف إلى أتمتة وتسهيل عملية إدارة الكفالات وربط جميع الأطراف المعنية في منظومة واحدة.

### الأهداف المحققة
- ✅ تقليل العمل اليدوي والتواصل التقليدي
- ✅ ربط الأطراف المعنية بسلاسة (مشرف، مورد، مندوب، متبرع)
- ✅ رفع الشفافية في تجربة المتبرع
- ✅ دعم اتخاذ القرار من خلال مؤشرات دقيقة
- ✅ توفير لوحة تحكم شاملة للمراقبة والتحليل

### البنية التحتية

#### Backend (Flask)
**الملفات الرئيسية:**
- `src/main.py` - الملف الرئيسي للخادم
- `src/models/user.py` - نماذج المستخدمين
- `src/models/sponsorship.py` - نماذج الكفالات
- `src/models/financial.py` - النماذج المالية
- `src/routes/auth.py` - مسارات المصادقة
- `src/routes/sponsorship.py` - مسارات الكفالات

**قاعدة البيانات:**
- SQLite مع SQLAlchemy ORM
- نماذج شاملة للمستخدمين والكفالات والمدفوعات
- علاقات محددة بين الجداول
- دعم للمصادقة والتفويض

#### Frontend (React)
**التقنيات المستخدمة:**
- React 19 مع Vite
- Tailwind CSS للتصميم
- Shadcn/UI للمكونات
- Lucide React للأيقونات
- Recharts للرسوم البيانية

**المكونات الأساسية:**
- `src/components/Header.jsx` - رأس الصفحة
- `src/components/HeroSection.jsx` - القسم الرئيسي
- `src/components/HowItWorks.jsx` - شرح آلية العمل
- `src/components/Footer.jsx` - تذييل الصفحة
- `src/components/ui/` - مكونات واجهة المستخدم

### طريقة التشغيل

#### تشغيل Backend:
```bash
cd saqya_platform
source venv/bin/activate
pip install -r requirements.txt
python src/main.py
```

#### تشغيل Frontend (للتطوير):
```bash
cd saqya-frontend
pnpm install
pnpm run dev
```

#### البناء والنشر:
```bash
cd saqya-frontend
pnpm run build
cp -r dist/* ../saqya_platform/src/static/
```
