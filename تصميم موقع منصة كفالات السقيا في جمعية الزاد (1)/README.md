# منصة كفالات السقيا - جمعية الزاد

هذا المشروع هو عبارة عن منصة إلكترونية متكاملة لإدارة كفالات السقيا لجمعية الزاد، ويشمل واجهة خلفية (Backend) مبنية باستخدام Flask وواجهة أمامية (Frontend) مبنية باستخدام React.

## محتويات المشروع

- `saqya_platform/`: مجلد الواجهة الخلفية (Backend) مبني بـ Flask.
- `saqya-frontend/`: مجلد الواجهة الأمامية (Frontend) مبني بـ React.
- `README.md`: هذا الملف الذي يحتوي على تعليمات التشغيل.
- `project_summary.md`: ملخص شامل للمشروع ومميزاته.
- `system_analysis.md`: وثيقة تحليل المتطلبات وتصميم النظام.
- `workflow_diagram.png`: مخطط تدفق العمليات.
- `ui_wireframes.md`: النماذج الأولية لواجهات المستخدم.
- `todo.md`: ملف تتبع المهام.

## المتطلبات الأساسية

تأكد من تثبيت البرامج التالية على جهازك:

- **Python 3.8+**
- **Node.js 18+**
- **pnpm** (مدير حزم Node.js، يمكن تثبيته عبر `npm install -g pnpm`)

## تعليمات التشغيل

اتبع الخطوات التالية لتشغيل المشروع على جهازك المحلي:

### 1. إعداد الواجهة الخلفية (Backend)

1.  **انتقل إلى مجلد الواجهة الخلفية:**
    ```bash
    cd saqya_platform
    ```

2.  **إنشاء بيئة افتراضية وتفعيلها:**
    ```bash
    python3 -m venv venv
    source venv/bin/activate  # لنظام Linux/macOS
    # venv\Scripts\activate  # لنظام Windows
    ```

3.  **تثبيت المكتبات المطلوبة:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **تشغيل الخادم:**
    ```bash
    python src/main.py
    ```
    سيتم تشغيل الخادم على المنفذ `5001` (أو المنفذ المحدد في `src/main.py`).

### 2. إعداد الواجهة الأمامية (Frontend)

1.  **انتقل إلى مجلد الواجهة الأمامية:**
    ```bash
    cd saqya-frontend
    ```

2.  **تثبيت المكتبات المطلوبة باستخدام pnpm:**
    ```bash
    pnpm install
    ```

3.  **بناء المشروع للإنتاج:**
    ```bash
    pnpm run build
    ```
    سيتم إنشاء مجلد `dist/` يحتوي على ملفات الواجهة الأمامية المبنية.

4.  **نسخ ملفات الواجهة الأمامية إلى الواجهة الخلفية:**
    بعد بناء الواجهة الأمامية، يجب نسخ الملفات الناتجة إلى مجلد `static` في الواجهة الخلفية. تأكد أنك في مجلد `saqya-frontend` ثم نفذ الأمر التالي:
    ```bash
    cp -r dist/* ../saqya_platform/src/static/
    ```
    **ملاحظة:** تأكد من حذف محتويات مجلد `../saqya_platform/src/static/` قبل النسخ لتجنب تداخل الملفات القديمة.

### 3. الوصول إلى المنصة

بعد تشغيل خادم الواجهة الخلفية ونسخ ملفات الواجهة الأمامية، يمكنك الوصول إلى المنصة عبر متصفح الويب على العنوان:

```
http://localhost:5001
```

## هيكل المشروع

### الواجهة الخلفية (saqya_platform)

- `src/`
  - `main.py`: نقطة الدخول الرئيسية للتطبيق.
  - `models/`: تعريف نماذج قاعدة البيانات (المستخدمين، الكفالات، الطلبات، إلخ).
  - `routes/`: تعريف مسارات API المختلفة (المصادقة، الكفالات، إلخ).
  - `static/`: مجلد يحتوي على ملفات الواجهة الأمامية المبنية (يتم نسخها هنا بعد البناء).
- `requirements.txt`: قائمة بجميع مكتبات Python المطلوبة.

### الواجهة الأمامية (saqya-frontend)

- `src/`
  - `App.jsx`: المكون الرئيسي للتطبيق.
  - `App.css`: ملف الأنماط الرئيسي.
  - `components/`: مجلد يحتوي على المكونات المختلفة (Header, Footer, AdminPanel, SupplierPanel, DonorPanel, RealSaqyaMap, إلخ).
  - `assets/`: مجلد للأصول مثل الصور (الشعار).
- `package.json`: تعريف المشروع ومكتبات Node.js المطلوبة.
